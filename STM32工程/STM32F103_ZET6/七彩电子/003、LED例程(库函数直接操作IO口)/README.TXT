
说明:
    此程序为七彩光子的太极M3-STM32F103开发板的配套程序,大部分程序已经进行封装或增加宏定义项,为的是方便大家快速移植到其他开发板;
    本程序只供学习使用，未经作者许可，不得用于其它任何用途！
    
开发环境:
    本程序使用Keil MDK5.14编写,部分程序中开启了C99标准。
    
程序打开方法:
    1、确认自己的电脑上已经安装了MDK5;
    2、找到工程目录中 PROJECT 目录中的STM32F103.uvprojx(文件名可能会根据工程不同而不同,但目录是确定的)双击即可打开。
